## Iniciando o projeto
1. Clone o repositório
 ```sh
   git clone https://github.com/wandskk/Monty-Hall.git
 ```
2. Vá até a pasta
 ```sh
   cd porta
 ```
3. Instale as dependências do projeto
 ```sh
   npm install or npm i
 ```
4. Execute o projeto
```sh
   npm start
 ```
5. Acesse o link
 ```sh 
    http://localhost:3000/
```
